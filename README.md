# zlldp
This is a LLDP Protocol library based on 802.1ab specification document and focus on,
- Stability
- Leak-Free
- Easy APIs

## Build Steps
```
$ make
```

## Run Demo
```
$ sudo LD_LIBRARY_PATH=../src tools/send <interface>
```

## Debug Steps
```
$ sudo LD_LIBRARY_PATH=../src gdb tools/send
```
